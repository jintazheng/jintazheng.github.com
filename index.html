Projects

Detail Enhancement Volume Rendering under Global Illumination
In this project, we built a platform for Volume Rendering in Real-time interaction, besides, it can enhance the detail information under the global illumination. Our platform contains an key light to offer the global illumination and a local light and shape volume to enhance details. In addition, this platform have a user-friendly interface and very easy to use. The platform is implemented by Qt, CUDA, OpenGL with C/C++.
Related algorithms: Global Illumination, Detail-enhancement, Volume Rendering, Real-time interaction.

Intelligent 3D ultrasound platform
In this project, we built a platform for 3D ultrasound rendering which can be used for the CT too. Our platform contains an improved photon mapping algorithm which can enhance the depth information and offer more realistic effect. More importantly, our method can achieve an real-time interaction. The Ambient Occlusion algorithm is also included in our platform for the high resolution volume data rendering. In addition, this platform have a user-friendly interface and very easy to use. The platform is implemented by Qt, CUDA, OpenGL with C/C++.
Related algorithms: Realstic Rendering, Ultrasound, Photon Mapping, Ambient Occlusion, Real-time interaction.
